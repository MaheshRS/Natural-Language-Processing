===== RUNNING VIA SCRIPT =====
There is one bash script available ('run.sh').
This script internally uses another script compile.sh, that compiles the source code. Once the compilation is complete, the run.sh script then runs the program with parameters passed to it.

1. run.sh (USAGE: "sh run.sh <event sequence>" or "bash run.sh <event sequence>")

   a. Copy and run the command 'sh run.sh <event sequence>' in the terminal.
   	  e.g. $sh run.sh '111222333222'

   or

   b. Copy and run the command 'bash run.sh <event sequence>' in the terminal.
   	  e.g. $bash run.sh '111222333222'


NOTE: 
1. PLEASE MAKE SURE THAT THE EVENT SEQUENCE IS CONTINIOUS WITHOUT ANY SPACE.
